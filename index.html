34py 주방리모델링  
[ ABOUT THIS PROJECT ]

화이트 컬러 벽과 따뜻한 우드마루를 시공하고, 트렌디한 디자인에 고급 마감재를 사용하여 심플하고 고급스런 내추럴모던 스타일로 완성했습니다.

거실과 주방, 멀티룸 등은이태리 수입 원목 쉐브론 마루를 시공하고, 각 방들은 강마루, 전체 벽은 실크 벽지로 도배했습니다. 각 공간에 스타일링한 가구와 거울 등은 공감디자인이 직접 제작했습니다.

30py 거실리모델링  
[ ABOUT THIS PROJECT ]

카페 인테리어  
[ ABOUT THIS PROJECT ]

다품종 전자동 시스템을 통해 원두 특성에 맞는 로스팅 및 블랜딩을 하여 최상의 커피 맛을 유지하는 라붐 커피만의 강점을 드러내고, 커피 전문 브랜드로서의 라붐 커피 만의 자유롭고 여유로운 분위기를 각인시킬 수 있도록 공간 컨셉을 “Creative Factory”로 잡았습니다. 또한 콘크리트와 철근, 가죽, 원목, 메탈 등의 마감재를 사용하여 빈티지한 공장의 모습들을 구현하고자 하였습니다.
