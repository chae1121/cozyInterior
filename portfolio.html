  • 분당아이파크 71평



    • 분당아이파크 71평  
    [ ABOUT THIS PROJECT ]

    우드와 웜그레이 컬러를 메인 컬러로 활용하고, 주방, 안방, 아이방, 서재 등 공간 별로 각각 좋아하는 컬러를 지정하여 포인트로 시공하는 등 내추럴 모던 컨셉으로 완성했습니다.
    타일과 우드의 조화를 통한 따뜻한 공간구성과 효과적인 동선구축, 자녀들 방의 공간 분리 및 공간별 수납공간 구성을 통해 높은 만족도를 보인 현장입니다.



  • 성복동 힐스테이트 56평



    • 성복동 힐스테이트 56평  
    [ ABOUT THIS PROJECT ]

    거실과 주방, 현관 등의 홀은 웜컬러의 웨인스코팅 월을 시공하고, 안방, 자녀방, 사모님 작업실 등 각각의 방은 심플한 디자인과 세련된 컬러로 모던하게 디자인하여 모던클래식 인테리어로 완성하였습니다.

    웜그레이 웨인스코팅으로 거실 아트월과 소파월, 복도 벽 하부를 시공하여 클래식하게 디자인했구요. 거실 우측의 주방 입구에는 단면을 웜그레이 클래식 몰딩으로 시공한 모서리가 둥근 사각 게이트를 신설하여 클래식한 분위기를 형성했습니다.



  • 정자동 파크뷰 48평



    • 정자동 파크뷰 48평  
    [ ABOUT THIS PROJECT ]

    화이트 컬러 벽과 따뜻한 우드마루를 시공하고, 트렌디한 디자인에 고급 마감재를 사용하여 심플하고 고급스런 내추럴모던 스타일로 완성했습니다.

    거실과 주방, 멀티룸 등은이태리 수입 원목 쉐브론 마루를 시공하고, 각 방들은 강마루, 전체 벽은 실크 벽지로 도배했습니다. 각 공간에 스타일링한 가구와 거울 등은 공감디자인이 직접 제작했습니다.



  • 브런치 카페 & 레스토랑



    • 브런치 카페 & 레스토랑  
    [ ABOUT THIS PROJECT ]

    Surfside 지역으로 유명한 하와이, 발리, 시드니, 캘리포니아 해변 로컬 식당들의 분위기에서 모티브를 구했고, 사용된 고재, 가구, 색감, 매장 곳곳에 걸린 서핑보드 등에서 이국적이고 자유분방한 에너지를 표현하고 있습니다.



  • 판교 여성병원



    • 판교 여성병원  
    [ ABOUT THIS PROJECT ]

    가장 익숙하고 편안한 공간을 떠올리면 으레 ‘집’을 생각하게 됩니다. 치료라는 목적에서 더 나아가 환자들이 편안히 있을 수 있는 공간을 만들기 위해 우리는 ‘따뜻한 작은 집’이라는 키워드를 설정했습니다. 이를 통해 병원이라는 장소가 주는 낯설고 차가운 느낌을 줄이고, 환자와 그들의 감정을 편안히 ‘이완’시킬 수 있는 공간을 계획하였습니다.



  • 카페 블렌드



    • 카페 블렌드  
    [ ABOUT THIS PROJECT ]

    다품종 전자동 시스템을 통해 원두 특성에 맞는 로스팅 및 블랜딩을 하여 최상의 커피 맛을 유지하는 라붐 커피만의 강점을 드러내고, 커피 전문 브랜드로서의 라붐 커피 만의 자유롭고 여유로운 분위기를 각인시킬 수 있도록 공간 컨셉을 “Creative Factory”로 잡았습니다. 또한 콘크리트와 철근, 가죽, 원목, 메탈 등의 마감재를 사용하여 빈티지한 공장의 모습들을 구현하고자 하였습니다.



  • 커리 레스토랑
